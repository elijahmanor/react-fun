# Introduction

## Repository

This workshop is intended to introduce React from the beginning and primarily focuses on using Function Components along with Hooks.

The [`react-fun`](https://github.com/elijahmanor/react-fun) repository has the code that goes along with this material.

Feel free to download the code and follow along on your computer or you can code online in [Codesandbox](https://codesandbox.io).

This documentation will provide instructions at certain key points in the workshop (either locally or via Codesandbox links) to catch up if you need them.

## Prerequisites

- Assumes basic HTML & CSS knowledge
- Assumes basic JavaScript (ES6+)
- Assumes basic knowledge about the DOM
- Assumes basic knowledge about Node & npm
