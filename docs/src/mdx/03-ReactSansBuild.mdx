# React Sans Build

## First Hello, World! App

For our first interactive lab, we will be using [Glitch](https://glitch.com/). It's a quick and easy way to play around with web technologies and build from others. Start a new `hello-webpage` project

Open the `index.html` file and inside your `<head>` element add the main `react` and `react-dom` scripts from [unpkg.com](https://unpkg.com/), a CDN that pulls from [npm](https://www.npmjs.com/) built by [Michael Jackson](https://twitter.com/mjackson). If your curious about unpkg, then you can check out [runpkg.com](https://runpkg.com/) built by [Formidable](https://formidable.com/blog/2019/runpkg/). Make sure these scripts are before the existing `<script src="/script.js" defer></script>` element.

```html
<script
  src="https://unpkg.com/react@16/umd/react.development.js"
  crossorigin
></script>
<script
  src="https://unpkg.com/react-dom@16/umd/react-dom.development.js"
  crossorigin
></script>
```

Now, inside of the `<body>` tag add a `<div>` element with `id` of `app`.

```html
<div id="app"></div>
<script>
  ReactDOM.render(
    React.createElement("h1", null, "Hello, World!"),
    document.getElementById("app")
  );
</script>
```

## `React.createElement` Signature

```js
React.createElement(
  type, // 1st argument
  [props], // 2nd optional argument
  [...children] // 3rd...nth optional children arguments
);
```

If you'd prefer you could pull out the script into another file to isolate what you are editing...

```html
<script src="./script.js"></script>
```

## Making things slightly Dynamic

You can start to make the contents dynamic using several techniques.

```javascript
const name = "Elijah";
ReactDOM.render(
  React.createElement("h1", null, `Hello, ${name}!`),
  document.getElementById("app")
);
```

The children can be multiple parameters...

```javascript
const name = "Elijah";
ReactDOM.render(
  React.createElement("h1", null, "Hello, ", name, "!"),
  document.getElementById("app")
);
```

The 2nd parameter represents props that are passed into the element. Note the use of `className` instead of `class` which is a reserved word.

```javascript
const name = "Elijah";
ReactDOM.render(
  React.createElement(
    "div",
    {
      title: name,
      className: "Ball",
    },
    "1"
  ),
  document.getElementById("app")
);
```

### `style.css`

```css
.Ball {
  display: inline-block;
  background-color: red;
  width: 50px;
  height: 50px;
  border-radius: 50%;
  line-height: 50px;
  text-align: center;
}
```

```javascript
const name = "Elijah";
ReactDOM.render(
  React.createElement(
    "div",
    {
      title: name,
      className: "Ball",
    },
    React.createElement("span", null, "1")
  ),
  document.getElementById("app")
);
```

```javascript
function Ball() {
  return React.createElement(
    "div",
    {
      title: name,
      className: "Ball",
    },
    React.createElement("span", null, "1")
  );
}
ReactDOM.render(React.createElement(Ball), document.getElementById("app"));
```
