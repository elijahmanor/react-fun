# Community Hooks

The following resources list a bunch of community custom hooks and other tutorials, talks, etc about React Hooks.

- [Collection of React Hooks](https://nikgraf.github.io/react-hooks/)
- [Awesome React Hooks](https://github.com/glauberfc/awesome-react-hooks)
- [useHooks](https://usehooks.com/)

## Lab

Replace the custom `useLocalStorage` hook with a community hook.

### Update `Background.js`

```shell
npm install crooks --save

# OR

yarn add crooks
```

```diff
- import { useLocalStorage } from "./customHooks";
+ import { useLocalStorage } from "crooks";
```
